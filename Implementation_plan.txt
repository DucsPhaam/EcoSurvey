```
Routes:
  POST /api/auth/forgot-password
  POST /api/auth/reset-password
```

- `frontend/index.html` — thêm `lang="vi"`, skip navigation link

**Checklist a11y cụ thể:**
```
□ Tất cả input có <label> liên kết
□ Error messages dùng aria-describedby
□ Focus visible trên tất cả interactive elements
□ Modal có focus trap + Escape close
□ Color contrast ≥ 4.5:1 cho text
□ Buttons có accessible name (không chỉ icon)
□ Survey form navigate được bằng Tab + Enter
□ Screen reader friendly (test với NVDA/VoiceOver)
□ Skip to main content link ẩn (hiện khi focus)
□ Slider cho grading có aria-valuemin/max/now/label
```

- `frontend/src/pages/auth/LoginPage.jsx` — nút "Đăng nhập bằng Google"

**Flow:**
```
User click "Login with Google"

→ Redirect frontend với token trong query param hoặc cookie

```
